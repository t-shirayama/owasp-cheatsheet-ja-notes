# Source Map

## 方針

- docs/asvs/ の章ページは、この対応表をもとに更新します。
- 同じ原文につき、翻訳ファイルを1つ作成し、公開用の英日対訳ページから参照します。
- 公式ページの URL、ローカルファイル、作成状況をここで追跡します。
- 英日対訳表示の Pilot 対応は [bilingual-map.md](bilingual-map.md) で追跡します。
