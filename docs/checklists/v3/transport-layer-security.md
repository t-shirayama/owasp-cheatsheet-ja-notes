## 開発チェックリスト

- [ ] TLS 1.2以上を有効化する。
- [ ] SSLと古いTLSを無効化する。
- [ ] 証明書期限を監視する。
- [ ] 証明書チェーンとホスト名検証を確認する。
- [ ] 外部診断でTLS設定を検証する。
