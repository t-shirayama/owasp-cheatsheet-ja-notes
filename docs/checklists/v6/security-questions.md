## 開発チェックリスト

- [ ] 秘密の質問の必要性を再評価する。
- [ ] 一般公開情報で答えられる質問を避ける。
- [ ] 回答を平文保存しない。
- [ ] 回答試行をレート制限する。
- [ ] MFAなどより強い代替を優先する。
