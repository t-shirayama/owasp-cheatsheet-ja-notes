## 開発チェックリスト

- [ ] resolverごとに認可を実装する。
- [ ] クエリ深さ制限を設定する。
- [ ] クエリ複雑度またはコスト制限を設定する。
- [ ] 本番で不要なイントロスペクションを制限する。
- [ ] GraphQLエラーに内部情報を含めない。
