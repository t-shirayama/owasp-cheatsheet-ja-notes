## 開発チェックリスト

- [ ] 全エンドポイントを一覧化する。
- [ ] 許可HTTPメソッドを制限する。
- [ ] リソースID改ざんをテストする。
- [ ] エラー応答に内部情報がないことを確認する。
- [ ] APIログとレート制限を確認する。
