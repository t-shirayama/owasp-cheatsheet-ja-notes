## 日本語訳

秘密の質問は推測、調査、漏えい、再利用に弱く、強い認証要素として扱うべきではありません。使う場合は限定的なリスク低減策が必要です。

## 主要な観点

- 秘密の質問を主要な認証要素にしない。
- 推測しにくく変化しにくい質問を選ぶ。
- 回答をパスワード同様に保護する。
