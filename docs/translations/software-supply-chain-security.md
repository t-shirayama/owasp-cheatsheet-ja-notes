## 日本語訳

ソフトウェアサプライチェーンは、依存関係、ビルド、CI/CD、署名、成果物、配布経路を含みます。改ざんや依存関係リスクを継続的に管理します。

## 主要な観点

- 依存関係とビルド成果物を追跡する。
- CI/CDの権限とシークレットを保護する。
- 署名やSBOMで成果物の信頼性を高める。
