## 日本語訳

暗号化ストレージは、保存データの機密性を守るために、何を暗号化するか、どこで暗号化するか、鍵をどう管理するかを設計する領域です。

## 主要な観点

- 保護対象データを分類する。
- 標準的で推奨される暗号方式を使う。
- 鍵管理を暗号化実装と同じくらい重視する。
