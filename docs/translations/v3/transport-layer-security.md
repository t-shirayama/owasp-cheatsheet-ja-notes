## 日本語訳

TLSは通信の機密性と完全性を守る基盤です。プロトコルバージョン、暗号スイート、証明書、HSTS、相互TLS、証明書検証を適切に設定します。

## 主要な観点

- 古いTLS/SSLバージョンを無効化する。
- 信頼できる証明書と正しいホスト名検証を行う。
- 強い暗号スイートと安全な設定を使う。
